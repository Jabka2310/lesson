# Telegram Chats Bot

Скрипт для создания супергрупп (чатов) в Telegram по запросу: задаёте количество чатов, дата подставляется автоматически, можно менять названия и добавлять участников по нику или номеру телефона.

Основан на идее [autoreg-telegram-channel](https://github.com/autoreg-telegram-channel): автоматизация через Pyrogram, но для **чатов (супергрупп)** и с интерактивным сценарием в терминале.

## Возможности

- **Создание N чатов** — указываете количество, создаются супергруппы.
- **Сегодняшняя дата** — подставляется в название по умолчанию (например: `Чат 2025-03-05 #1`).
- **Редактирование названий** — до создания (ввод своих названий) и после (переименование созданных чатов).
- **Добавление участников** — по нику в Telegram (`@username`) или по номеру телефона (`+79991234567`), через запятую.

## Установка

1. Клонируйте или скопируйте папку `telegram-chats-bot`.
2. Создайте виртуальное окружение и установите зависимости:

```bash
cd telegram-chats-bot
python3 -m venv venv
source venv/bin/activate   # Windows: venv\Scripts\activate
pip install -r requirements.txt
```

3. Получите API-ключи на [my.telegram.org](https://my.telegram.org) (приложение типа "Desktop").
4. Скопируйте пример конфига и заполните:

```bash
cp .env.example .env
# Отредактируйте .env: API_ID, API_HASH, при желании SESSION_NAME
```

## Запуск

### Вариант 1: Терминал (интерактивно)

```bash
python chats_bot.py
```

При первом запуске потребуется войти в аккаунт Telegram (код из приложения или пароль двухфакторной аутентификации). Сессия сохраняется в файл (по умолчанию `chats_bot_session.session`).

### Вариант 2: Через бота в Telegram

**Почему «просто ботом» нельзя:** Telegram Bot API не умеет создавать чаты/группы — у ботов нет таких прав. Поэтому делается **гибрид**: бот принимает команды, а создание выполняет твой аккаунт (user client через Pyrogram).

1. Создай бота в [@BotFather](https://t.me/BotFather), получи токен.
2. В `.env` добавь: `BOT_TOKEN=твой_токен`.
3. Запусти:

```bash
python bot_runner.py
```

Один раз войди в аккаунт (как в варианте 1), дальше бот будет работать. В личку боту можно писать:
- `/start` — справка
- `/date` — сегодняшняя дата
- `/create 5` — создать 5 чатов с названиями по умолчанию
- `/create 3 @user1 +79001234567` — создать 3 чата и добавить участников

## Пример сценария

1. Запуск → ввод кода/пароля при первом входе.
2. **Сколько чатов создать?** — например `3`.
3. **Названия** — для каждого чата своё или Enter (будет "Чат 2025-03-05 #1", "#2", "#3").
4. **Участники** — например: `@user1, +79001234567, user2` (ник без @ допишется автоматически).
5. Скрипт создаёт чаты, добавляет участников в каждый.
6. **Изменить названия?** — при ответе `y` можно переименовать уже созданные чаты.

## Переменные окружения (.env)

| Переменная     | Описание |
|----------------|----------|
| `API_ID`       | Числовой ID приложения с my.telegram.org |
| `API_HASH`     | Хеш приложения |
| `SESSION_NAME` | Имя файла сессии пользователя (без расширения) |
| `BOT_TOKEN`    | Токен бота от @BotFather (только для `bot_runner.py`) |

## Ограничения Telegram

- Лимиты на создание чатов и добавление участников; при срабатывании лимита скрипт ждёт и повторяет (FloodWait).
- Добавление по номеру телефона возможно, если номер в контактах или пользователь доступен для приглашения.
